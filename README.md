# 🎓 AI Learning Hub

An Intelligent E-Learning Recommendation System built using Flask, Python, SQLite, Bootstrap and Machine Learning.

## 🚀 Features

- User Authentication (Login/Register)
- Admin Dashboard
- User Management
- Course Management (CRUD)
- AI Course Recommendation
- Analytics Dashboard
- Profile Management
- Dark Mode
- Responsive UI
- Charts using Chart.js
- Search Functionality
- Loading Spinner
- Toast Notifications
- Custom 404 Error Page

---

## 🛠 Tech Stack

- Python
- Flask
- SQLite
- SQLAlchemy
- Bootstrap 5
- HTML
- CSS
- JavaScript
- Chart.js
- Scikit-Learn
