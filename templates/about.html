🎓 About EduAI

EduAI is an Intelligent E-Learning Recommendation System that helps learners discover the most relevant courses based on their interests.

🚀 Project Features

  • 🔐 User Registration & Login
  • 🤖 AI-Based Course Recommendation
  • 🔍 Course Search
  • 📚 Course Management using Database
  • ⭐ Personalized Recommendations using TF-IDF & Cosine Similarity

🛠 Technologies Used

🐍 Python
🌐 Flask
💾 SQLite
🎨 Bootstrap
